📦 Setup Instructions

    
To run the Python backend on your machine:

    
# 1. Install Python 3.11+
# 2. Install Java 21 (required for Gradle)
# 3. Install Gradle 8+ (for building mods)

# 4. Clone/download the mc_mod_ai folder

# 5. Install Python dependencies (no pip packages needed — uses stdlib only!)
#    Optional: pip install requests  (for better HTTP handling)

# 6. Set API keys (or use --flags)
export GEMINI_API_KEY="your_google_ai_studio_key"
export GROQ_API_KEY="your_groq_key"
export JULES_API_KEY="your_jules_key"  # same as Gemini key

# 7. Run!
python main.py "A magic sword that shoots lightning" --name "Thunder Blade"
python main.py "A new ore that drops experience" --output ./my_mods
